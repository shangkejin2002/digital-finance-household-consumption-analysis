# Data

本プロジェクトで使用した元データは、利用制限およびプライバシー保護の観点から、このリポジトリには含めていません。

分析を再現する場合は、以下の形式でデータファイルを `data/` フォルダに配置してください。

```text
data/analysis_data.xlsx
```

## 想定される主な変数

* `ID`：世帯 ID
* `Province`：省・地域名
* `DFI`：2014 年時点の省別デジタル金融包摂指数
* `Treat_p`：高 DFI 地域を示す二値変数
* `FOOD_2014`：2014 年の食品支出
* `FOOD_2018`：2018 年の食品支出
* `TOTAL_2014`：2014 年の総消費支出
* `TOTAL_2018`：2018 年の総消費支出
* `Engel_2014`：2014 年のエンゲル係数
* `Engel_2018`：2018 年のエンゲル係数

## 注意

本リポジトリの R スクリプトでは、以下のパスからデータを読み込む設定にしています。

```r
read_excel("data/analysis_data.xlsx", sheet = "Sheet1")
```
