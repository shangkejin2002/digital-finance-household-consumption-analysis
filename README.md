# Digital Financial Inclusion and Food Consumption in China

## プロジェクト概要

本プロジェクトは、卒業論文「デジタル金融包摂が中国の食料消費に与える影響」をもとに作成したデータ分析プロジェクトです。

中国の家計パネル調査である CFPS の 2014 年・2018 年データと、省別のデジタル金融包摂指数（DFI）を結合し、DFI の発展水準が家計のエンゲル係数にどのような影響を与えるのかを分析しました。

分析手法としては、差分の差分法（Difference-in-Differences, DID）を用い、世帯固定効果および年次固定効果を考慮した推定を行っています。

## 使用データ

本研究では、以下のデータを使用しました。

* **CFPS（China Family Panel Studies）**

  * 2014 年および 2018 年の家計レベルデータ
  * 家計の食品支出、総消費支出、地域情報などを使用

* **DFI 指数**

  * 北京大学デジタル金融研究センターが作成した省別デジタル金融包摂指数
  * 本分析では、2014 年時点の省別 DFI 指数を地域のデジタル金融発展水準として使用

* **主な被説明変数**

  * エンゲル係数
  * 食品支出を総消費支出で割った値として定義

## 分析方法

本プロジェクトでは、デジタル金融包摂の発展が家計のエンゲル係数に与える影響を検証するため、差分の差分法（Difference-in-Differences, DID）を用いました。

主な推定モデルでは、2014 年時点の省別 DFI 指数と、政策後を表す Post ダミー変数の交互項を用いて、DFI 水準の高い地域ほど政策後にエンゲル係数がどのように変化したかを分析しました。

また、世帯固定効果と年次固定効果を導入することで、世帯ごとの時間に依存しない特徴や、全国共通のマクロ的な変化を統制しました。

## 使用した技術

本プロジェクトでは、主に以下のツール・ライブラリを使用しました。

* **R**

  * データ整理、記述統計、回帰分析、可視化に使用

* **tidyverse**

  * データの整形、集計、可視化に使用

* **fixest**

  * 固定効果を含む DID 推定に使用

* **readxl**

  * Excel 形式のデータ読み込みに使用

* **modelsummary**

  * 回帰結果の表作成に使用

* **ggplot2**

  * DFI 区間別のエンゲル係数の可視化に使用
